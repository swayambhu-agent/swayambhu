Mind Tab — What should it show?

The Mind tab is the deep dive into the agent's cognitive state. What sections does it need?

S

Samskaras

All samskaras sorted by strength. Each shows pattern text and a strength bar (0-1). Color-coded: strong (green), medium (amber), weak/eroding (red). Click to expand and see which sessions tested it, recent strength trajectory.

D

Desires

Active desires with direction (approach ↑ / avoidance ↓). Shows source principles, when last evolved by D operator. Visual distinction between approach and avoidance vectors.

E

Experience Stream

Recent experiences chronologically. Each shows surprise score, salience, narrative snippet. Sparkline of salience over time — are experiences getting more or less notable? Bootstrap indicator if ε is empty.

O

Operator Health

When did S and D last run? What did they produce? Session count since last deep-reflect. Next DR scheduled for when. Did the last DR create/deepen/erode any samskaras? Did D evolve any desires? Bootstrap status (has the agent earned its first samskaras/desires?).

Δ

Dynamics / Timeline

A unified timeline showing cognitive events: samskara created, samskara strength changed, desire evolved, experience recorded, deep-reflect ran. Like the session timeline but for the mind, not for actions. Shows the cognitive system developing over time.
